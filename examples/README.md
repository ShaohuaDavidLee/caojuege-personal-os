# 样例输出

这个目录里放的是 `司马迁.skill` 跑完之后会产出什么样的文件。

样例是创作者本人愿意公开的版本，**不是 skill 默认会输出的完整文件**。默认输出里会包含更具体的事实、判断和边界——这些通常只放在私有 context 里使用，不上传到任何公开仓库。

## 当前样例

- [`david-persona-agent.public.md`](./david-persona-agent.public.md)：草诀歌 AI Labs 主理人 David 的 persona 公开版。

## 我的输出是默认私有的吗？

是的。这个 skill 不会自动公开任何东西。生成的两份资产（`persona-agent.md` 和 `personal-homepage.html`）默认都只写到本地。是否、以及以什么粒度公开，由你自己决定。

样例文件的存在只是为了给新用户一个直观参照。
